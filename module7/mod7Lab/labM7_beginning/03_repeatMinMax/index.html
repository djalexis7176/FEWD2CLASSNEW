repeat()

1. Basic repeat() Usage

Instead of writing:

grid-template-columns: 100px 100px 100px 100px;
You can simplify it using:
grid-template-columns: repeat(4, 100px);

6. Combining repeat() with minmax()

You can use minmax() inside repeat() to create columns that have both a minimum and a maximum size, ensuring better responsiveness.

grid-template-columns: repeat(2, minmax(100px, 1fr)) 200px;
