Understanding Relative Units

Relative units are essential for creating fluid layouts that adapt to different screen sizes.

Commonly Used Relative Units:

  • Percentage (%): Width: 50% sets the width to 50% of the parent container.
  • em: This unit is relative to the font size of the element it is used in. If you set a font size in em, it is calculated based on the font size of the parent element. Thus, if the font size of a parent is changed, the child elements using em will also change in relation to that size.
  • rem: This unit stands for "root em" and is relative to the font size of the root element (). This means that it provides a more consistent and predictable sizing since it does not depend on the font size of the parent element.
  • Viewport Height (vh): Height: 50vh sets the height to 50% of the viewport height.
  • Viewport Width (vw): Width: 100vw makes the element as wide as the viewport.
